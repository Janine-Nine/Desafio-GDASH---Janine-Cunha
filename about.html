Sobre GDASH

Visão Geral

GDASH é uma plataforma completa para monitoramento de geração de energia fotovoltaica. O sistema coleta dados climáticos em tempo real e fornece insights baseados em IA para otimizar a produção de energia renovável.

Tecnologias

  • Frontend: React + Vite + Tailwind CSS
  • Backend: Express.js + TypeScript
  • Banco de Dados: PostgreSQL
  • Cache: Redis
  • Coleta de Dados: Python
  • Containerização: Docker

Funcionalidades

  • Dashboard de monitoramento em tempo real
  • Geração automática de insights de IA
  • Exportação de dados em CSV/XLSX
  • Gerenciamento de usuários
  • Integração com APIs externas
