Dashboard API Sobre

Documentação da API

Autenticação

A API utiliza autenticação via JWT (JSON Web Tokens).

POST /api/auth/login
Content-Type: application/json

{
  "email": "admin@gdash.com",
  "password": "123456"
}

Endpoints - Clima

Listar Logs Climáticos

GET /api/weather/logs

Criar Log Climático

POST /api/weather/logs
Authorization: Bearer {token}
Content-Type: application/json

{
  "timestamp": "2025-01-01T12:00:00Z",
  "location": "Pelotas, RS, BR",
  "temperature": 22.3,
  "humidity": 78,
  "windSpeed": 12,
  "condition": "Cloudy",
  "precipitationProb": 40
}

Exportar Dados

GET /api/weather/export.csv
GET /api/weather/export.xlsx

Endpoints - Insights

Listar Insights

GET /api/insights

Gerar Novos Insights

POST /api/insights/generate
Authorization: Bearer {token}

Análise de Eficiência

GET /api/insights/efficiency

Endpoints - Usuários

Listar Usuários

GET /api/users
